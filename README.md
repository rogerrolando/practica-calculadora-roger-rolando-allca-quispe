# Calculadora Web - Roger Rolando Allca Quispe

## Descripción
Calculadora web funcional desarrollada con HTML, CSS y JavaScript.

## Características
- Operaciones básicas (suma, resta, multiplicación, división)
- Diseño responsive y moderno
- Soporte para teclado físico
- Manejo de errores
- Botón de igual de doble altura

## Enlace de la aplicación
https://rogerrolando.github.io/practica-calculadora-roger-rolando-allca-quispe

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript
- GitHub Pages
